# Chatter

Chatter is a real-time web chat application that allows users to
create and join chat rooms to send messages publicly and privately to other users in the chat room.

Uses:
- [Node.js (Backend)](https://nodejs.org/en/)
- [Express (Frontend)](https://expressjs.com/)
- [MongoDB (Database)](https://mongodb.com/)
- [Socket.IO (Client/server communication)](https://socket.io/)
- [Heroku (App deployment)](https://www.heroku.com/home)
